# API Contract: Certifications

**Service file**: `src/features/certifications/services/certifications.service.ts`
**Pattern**: Supabase SDK direct queries, ServiceResult<T> return type

## Queries

### getCertificationsForStudent(studentId: string)
- **Table**: `certifications`
- **Filter**: `student_id = studentId`, `status IN ('issued', 'revoked')`
- **Select**: `*, program:programs(name, name_ar), track:program_tracks(name, name_ar), teacher:profiles!certifications_teacher_id_fkey(full_name, display_name)`
- **Order**: `issue_date DESC`
- **Used by**: Student certificates tab

### getCertificationById(certificationId: string)
- **Table**: `certifications`
- **Select**: Full row with joined program, track, teacher, issuer profiles
- **Used by**: Certificate detail screen, certificate image rendering

### getCertificationRequests(programId: string, status?: string)
- **Table**: `certifications`
- **Filter**: `program_id = programId`, optional status filter
- **Select**: `*, student:profiles!certifications_student_id_fkey(full_name, display_name, avatar_url), teacher:profiles!certifications_teacher_id_fkey(full_name)`
- **Order**: `created_at DESC`
- **Used by**: Supervisor review queue, program admin issuance queue

## Mutations

### recommendForCertification(input: RecommendInput)
- **Table**: `certifications`
- **Operation**: INSERT
- **Fields**: `student_id, program_id, track_id, enrollment_id, type, title, title_ar, teacher_id, recommended_by, status: 'recommended'`
- **Validation**: Call `get_certification_eligibility(enrollment_id)` first
- **Used by**: Teacher student detail → "Recommend for Certification" button

### reviewCertification(certificationId: string, action: 'approve' | 'reject', note?: string)
- **Table**: `certifications`
- **Operation**: UPDATE
- **Fields**: If approve: `status → 'supervisor_approved', supervisor_id`. If reject: `status → 'rejected', rejection_reason = note (required)`
- **Used by**: Supervisor certification review

### issueCertification(certificationId: string, chainOfNarration?: string)
- **Table**: `certifications`
- **Operation**: UPDATE
- **Fields**: `status → 'issued', issued_by, issue_date = now(), chain_of_narration, certificate_number (auto-generated by trigger)`
- **Used by**: Program admin issuance

### revokeCertification(certificationId: string, reason: string)
- **Table**: `certifications`
- **Operation**: UPDATE
- **Fields**: `status → 'revoked', revocation_reason = reason, revoked_by`
- **Side effect**: Notify the student (certificate now marked as revoked in their view) and the recommending teacher
- **Used by**: Program admin revocation

## Edge Function: verify-certificate

- **Path**: `/functions/v1/verify-certificate/{certificate_number}`
- **Method**: GET
- **Auth**: None (public, `verify_jwt: false`)
- **Response**: HTML page with certificate details, "revoked" warning if applicable, or "not found" error for invalid numbers
- **Query**: `certifications` table by `certificate_number`, joined with student/program/teacher names
- **Rate limit**: 10 requests/minute per IP to prevent enumeration
