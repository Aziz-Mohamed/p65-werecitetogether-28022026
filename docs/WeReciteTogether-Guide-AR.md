# القسم الأول: فهم التطبيق

---

## ما هو تطبيق نتلو سوياً؟

نتلو سوياً هو التطبيق الذي بنيناه ليحلّ محل قنوات التلجرام والتتبع اليدوي. بدلًا من الرسائل المتفرقة والجداول والملاحظات الصوتية — كل شيء الآن في مكان واحد: الحصص، التقدم، الحضور، التقييمات، والتقارير.

عند تسجيل الدخول، يتعرّف التطبيق على دورك ويعرض لك الشاشات والأدوات المناسبة تمامًا.

---

## الأدوار الخمسة

نظامنا يتكون من خمسة أدوار. كل دور يرى لوحة تحكم مختلفة ولديه صلاحيات مختلفة:

```
        المدير العام (Master Admin)
               │
               │ يُشرف على كل شيء
               ▼
        مدير البرنامج (Program Admin)
               │
               │ يدير برنامجًا محددًا
               ▼
           المشرف (Supervisor)
               │
               │ يتابع المعلّمين المُسندين إليه
               ▼
           المعلّم (Teacher)
               │
               │ يُعلّم ويُقيّم
               ▼
           الطالب (Student)
```

كل مستوى يرى ما تحته لكن ليس ما فوقه. المشرف يرى معلّميه وطلابهم — لكنه لا يرى فرق المشرفين الآخرين.

---

## كيف ننظّم الأمور

```
البرنامج (Program)
  └── المسار (Track) — مسلك تعليمي داخل البرنامج
        └── الحلقة (Class) — مجموعة طلاب مع معلّم
              └── تسجيل الطالب (Enrollment)
```

**أنواع البرامج:**
- **مجاني**: تسجيل مفتوح، انضم فورًا
- **منظّم**: مسارات رسمية بسعة محددة، قد يتطلب موافقة
- **مختلط**: فيه أقسام مجانية وأخرى منظّمة

---

## المفاهيم الأساسية

### الحصص
لقاء تعليمي يُقيّم فيه المعلّم الطالب في ثلاث درجات (١–٥ لكل منها):
- **الحفظ** — دقة الاسترجاع
- **التجويد** — تطبيق قواعد التلاوة
- **القراءة** — الطلاقة والأداء

### تتبع الحفظ
القرآن مُقسَّم إلى ٢٤٠ ربعًا. الطالب يبني حفظه ربعًا بربع. كل ربع يمر بمراحل: **قيد الحفظ ← جاهز للإجازة ← مُجاز**.

### نظام المراجعة
الأرباع المُجازة تُتابَع بنظام التكرار المتباعد:
- **طازج** — مُراجَع حديثًا، محفوظ جيدًا
- **يتلاشى** — يضعف، راجعه قريبًا
- **معرّض** — يحتاج انتباهًا
- **عاجل** — متأخر، أعطه الأولوية
- **خامل** (٩٠+ يوم) — يحتاج إعادة إجازة كاملة

يتلقى الطلاب **واجبات مراجعة** يومية للحفاظ على حفظهم.

### الإجازة
عندما يُتقن الطالب قسمًا: المعلّم يوصي ← المشرف يوافق ← تُصدَر الشهادة.

### التحفيز
- **الملصقات** — يمنحها المعلّمون (برونزية، فضية، ذهبية، ماسية)
- **الأوسمة** — تُكتسب تلقائيًا عند الإنجازات
- **سلسلة الإنجاز** — أيام النشاط المتتالية
- **لوحة المتصدّرين** — ترتيب حسب المستوى والسلسلة

---

## البدء

١. افتح التطبيق وسجّل الدخول بـ **جوجل** أو **أبل**
٢. أول مرة؟ أكمل ملفك الشخصي (الاسم، اللغة، نبذة اختيارية)
٣. التطبيق ينقلك تلقائيًا للوحة تحكم دورك
٤. فعّل الإشعارات عند السؤال لتبقى مطّلعًا

# القسم الثاني: الدليل حسب كل دور

---

## الفصل الأول: الطالب (طالب)

### تبويباتك

| التبويب | ماذا يفعل |
|---------|-----------|
| **لوحة التحكّم** | الحصص القادمة، السلسلة، الإحصائيات، الواجبات |
| **البرامج** | تصفّح البرامج والانضمام إليها |
| **الحفظ** | تتبع تقدمك في الأرباع |
| **المراجعة** | إدارة صحة المراجعة والواجبات |
| **رحلتي** | خريطة القرآن، مجموعة الملصقات، مشاركة التقدم |
| **الملف الشخصي** | الإعدادات والتفضيلات |

### الإجراءات الرئيسية

**التسجيل في برنامج:**
١. تبويب البرامج ← تصفّح ← اضغط على برنامج
٢. مجاني: اضغط "انضمام"
٣. منظّم: اختر مسارًا ← اختر حلقة ← اضغط "تسجيل"
٤. إذا ممتلئ، انضم لقائمة الانتظار

**تتبع الحفظ:**
- معلّمك يُعيّن حفظًا جديدًا ← يظهر كواجب في لوحة التحكم
- ادرس ← اقرأ على معلّمك ← المعلّم يُجيز الربع ← يدخل دورة المراجعة

**إدارة المراجعة:**
- تفقّد تبويب المراجعة يوميًا لرؤية واجباتك
- بعد التسميع لمعلّمك، اضغط "تم التسميع"
- ربع خامل ٩٠+ يومًا يحتاج إعادة إجازة

**سجل الحصص:**
- لوحة التحكم ← "عرض الحصص" ← الدرجات والملاحظات والتفاصيل لكل حصة

**المعلّمون المتاحون:**
- "المتاحون الآن" من لوحة التحكم تُظهر المعلّمين المتاحين مع روابط الاجتماع

**فعاليات الهِمَم:**
- عند الإعلان، اختر مسارك (٣–٣٠ جزءًا)، حدد أرقام الأجزاء، اختر أوقات الصلاة، سجّل

## الفصل الثاني: المعلّم (معلّم)

### تبويباتك

| التبويب | ماذا يفعل |
|---------|-----------|
| **لوحة التحكّم** | حصص اليوم، الطلاب، التوفر، التقييمات |
| **الطلاب** | طلابك المُسندون |
| **الحصص** | الحصص القادمة والسابقة |
| **تقدم الحلقة** | تحليلات الأداء |
| **الملف الشخصي** | التوفر، رابط الاجتماع |

### الإجراءات الرئيسية

**تسجيل حصة:**
١. لوحة التحكم ← "تسجيل حصة" (أو من تبويب الحصص)
٢. اختر الطالب ← حدد التاريخ
٣. قيّم: الحفظ، التجويد، القراءة (١–٥ لكل بُعد)
٤. أضف ملاحظات ← أرفق مقطعًا صوتيًا اختياريًا ← احفظ

**تشغيل حصة مجدولة:**
١. الحصص ← القادمة ← اضغط الحصة ← "بدء الحصة"
٢. سجّل الحضور (حاضر/غائب/متأخر/معذور)
٣. قيّم كل طالب ← أنهِ الحصة

**منح ملصق:**
١. "منح ملصق" من لوحة التحكم أو ملف الطالب
٢. اختر الطالب ← اختر الملصق ← سبب اختياري ← أكّد

**إجازة ربع:**
١. شاشة حفظ الطالب ← ابحث عن ربع "جاهز للإجازة"
٢. اضغط "إجازة" ← أكّد

**مراجعة ربع مُجاز:**
- "جيد" ← الطزاجة تعود لـ ١٠٠٪
- "ضعيف" ← الطزاجة تعود لـ ٥٠٪
- خامل ٩٠+ يومًا ← يحتاج إعادة إجازة

**إدارة التوفر:**
١. الملف الشخصي ← التوفر ← فعّل/عطّل لكل برنامج
٢. حدد رابط الاجتماع (جوجل ميت، زوم، إلخ) في الملف الشخصي

**تعيين واجبات:**
١. ملف الطالب ← "تعيين حفظ جديد"
٢. اختر السورة ونطاق الآيات والنوع وتاريخ التسليم ← أنشئ

**تسجيل الحضور (إن فُعّل):**
- لوحة التحكم ← "تسجيل الحضور" ← يُتحقق عبر GPS/WiFi
- خارج المنطقة؟ "طلب استثناء" مع السبب
- نهاية الوقت ← "تسجيل الانصراف"

## الفصل الثالث: المشرف (مشرف)

### تبويباتك

| التبويب | ماذا يفعل |
|---------|-----------|
| **الرئيسية** | الإحصائيات، التنبيهات، الإجراءات السريعة |
| **المعلمون** | المعلّمون المُسندون إليك |
| **التقارير** | تحليلات أداء المعلّمين |
| **الملف الشخصي** | ملفك وبرامجك |

### الإجراءات الرئيسية

**متابعة فريقك:**
- الرئيسية تُظهر: عدد المعلّمين، عدد الطلاب، جلسات الأسبوع
- تنبيه عدم النشاط يُحذّر عن معلّمين لم يُنفذوا حصصًا هذا الأسبوع

**مراجعة معلّم:**
١. تبويب المعلمون ← اضغط على معلّم
٢. شاهد: سجل الحصص، قائمة الطلاب، مقاييس الأداء
٣. اضغط على أي طالب لرؤية تقدمه

**الإبلاغ عن مشكلة:**
١. تفاصيل المعلّم ← "الإبلاغ عن مشكلة"
٢. صف المخاوف (حتى ٥٠٠ حرف)
٣. "إرسال التنبيه لمديري البرنامج"

**إعادة تعيين طالب:**
١. تفاصيل المعلّم ← الطلاب ← اختر طالبًا ← "إعادة تعيين"
٢. اختر المعلّم الجديد ← أكّد

**طلبات الإجازة:**
- الرئيسية ← "طلبات الإجازة" ← راجع الطلبات ← وافق أو اطلب مزيدًا من التقييم

**لوحة المكافآت:**
- إحصائيات توزيع الملصقات، أنشط المعلّمين في المكافآت، توزيع الأوسمة

**التقارير المتاحة:**
- الحصص لكل معلّم، توزيع الطلاب، متوسط التقييم، تنبيهات عدم النشاط

## الفصل الرابع: مدير البرنامج (مدير البرنامج)

### تبويباتك

| التبويب | ماذا يفعل |
|---------|-----------|
| **الرئيسية** | إحصائيات البرنامج والتنبيهات |
| **الحلقات** | إدارة الحلقات |
| **الفريق** | إدارة المعلّمين والمشرفين والإداريين |
| **التقارير** | تحليلات العبء والتقدم |
| **الإعدادات** | تهيئة البرنامج |
| **الملف الشخصي** | ملفك |

*إذا تدير عدة برامج، ستختار أيها تدير أولًا. بدّل في أي وقت بزر "تبديل البرنامج".*

### الإجراءات الرئيسية

**إنشاء حلقة:**
١. تبويب الحلقات ← "إنشاء حلقة"
٢. أدخل الاسم، الحد الأقصى للطلاب، عيّن معلّمًا، اختر مسارًا ← أنشئ

**إدارة التسجيل:**
- الطلبات المعلّقة تظهر كتنبيهات في الرئيسية
- اضغط للمراجعة ← وافق أو ارفض كل طلب
- "الموافقة على الكل" للموافقة الجماعية
- قائمة الانتظار: "ترقية التالي" تعرض المكان (الطالب لديه ٤٨ ساعة للقبول)

**إدارة الفريق:**
١. تبويب الفريق ← "إضافة عضو"
٢. ابحث عن مستخدم ← حدد الدور (معلّم، مشرف، أو مدير برنامج)
٣. ربط مشرف بمعلّم: ابحث عن المعلّم ← "ربط المشرف" ← اختر المشرف
٤. إزالة: اختر العضو ← "إزالة" (تحذير إذا لديه طلاب نشطون)

**إعدادات البرنامج:**

| الإعداد | ماذا يتحكم فيه |
|---------|----------------|
| الحد الأقصى للطلاب لكل معلّم | سعة المعلّم |
| حد الحصص المجانية اليومي | الحد الأقصى للحصص المجانية يوميًا |
| حد إشعار قائمة الانتظار | متى يُنبَّه عن طول القائمة |
| حدود التقييم | تحديد مستويات الأداء الجيد / التحذيري / المقلق |

**التقارير المتاحة:**
- توزيع عبء عمل المعلّمين، تقدم الطلاب، اتجاهات تكرار الحصص

## الفصل الخامس: المدير العام (المدير العام)

### أقسامك

بدون تبويبات — تصفح كامل لجميع الأقسام: لوحة التحكم، المستخدمون، البرامج، الحلقات، المعلّمون، الطلاب، الملصقات، الحضور، التقارير، الإعدادات، الإجازات.

### الإجراءات الرئيسية

**نظرة عامة:**
- إجمالي الطلاب، المعلّمين، الحصص النشطة، عدد البرامج
- إجراءات سريعة: إضافة حلقة، إنشاء برنامج، إدارة المستخدمين

**إدارة المستخدمين:**
١. المستخدمون ← ابحث بالاسم أو البريد ← اضغط على المستخدم
٢. تغيير الدور: "تغيير الدور" ← اختر ← أكّد
٣. تعيين دور برنامجي: "تعيين دور برنامجي" ← اختر البرنامج والدور
٤. ترقية لمدير عام: "ترقية إلى مدير عام" ← أكّد
   - *لا يمكن إزالة آخر مدير عام*

**إنشاء برنامج:**
١. البرامج ← "إنشاء برنامج"
٢. الاسم (عربي + إنجليزي)، الوصف، الفئة (مجاني/منظّم/مختلط)
٣. الإعدادات: موافقة تلقائية، حد الطلاب/معلّم، مدة الحصة
٤. أضف مسارات عند الحاجة

**إدارة كتالوج الملصقات:**
١. الملصقات ← "إنشاء ملصق"
٢. اختر أيقونة (٦ فئات: العبادة، السماوية، العمارة، الطبيعة، الإنجاز، الخط العربي)
٣. حدد الاسم (عربي + إنجليزي)، الفئة ← احفظ

**حضور المعلّمين:**
١. الحضور ← شاهد تسجيلات اليوم
٢. الاستثناءات المعلّقة: راجع السبب ← وافق أو ارفض

**إعداد التحقق:**
١. الإعدادات ← إعدادات التحقق
٢. حدد إحداثيات GPS + نطاق السياج الجغرافي (٥٠–٢٠٠٠ متر)
٣. حدد اسم شبكة WiFi
٤. الطريقة: GPS فقط / WiFi فقط / كلاهما / أيّ منهما

**إعدادات المنصة:**
- اسم المنصة (عربي + إنجليزي)، منصة الاجتماع الافتراضية، ساعات الهدوء، صلاحيات المعلّمين

**إنشاء مستخدمين:**
- طلاب: الاسم، اسم المستخدم، كلمة المرور (٦+ أحرف)، حلقة اختيارية
- معلّمون: الاسم، اسم المستخدم، كلمة المرور ← ثم عيّنهم للبرامج/الحلقات

**إعادة تعيين كلمات المرور:**
- الإعدادات ← "إعادة تعيين كلمة المرور" ← اختر المستخدم ← أدخل كلمة المرور الجديدة

**التقارير المتاحة:**
- اتجاهات التسجيل، حجم الحصص حسب البرنامج، نشاط المعلّمين، حضور المعلّمين، تقدم الحفظ، معدلات إتمام الحصص
