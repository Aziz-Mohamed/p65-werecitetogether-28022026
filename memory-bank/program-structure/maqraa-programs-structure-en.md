# Maqra'a Structure

## Division One: Open Division
> (Open to join at any time — no fixed schedules or organized classes)

### 1. Rotating Recitation (Al-Tasmee' bil-Tanawub)
- **Matn (Texts) Section**
- **Quran Section**
  - Certified teachers (Mujazeen)
  - Students

### 2. Non-Arab Students (Al-A'ajim)
- Pronunciation correction

---

## Division Two: Structured Division
> (Fixed classes with set schedules and a defined number of students per class)

### 1. Children
- Talqeen (direct oral instruction)
- Al-Qa'ida Al-Nooraniya
- Memorization track

### 2. Qira'at (Recitation Narrations)
- Riwayat Hafs 'an 'Asim
- Riwayat Warsh 'an Nafi'
- Riwayat Qaloon 'an Nafi'
- Riwayat Al-Duri 'an Abi 'Amr

### 3. Mutoon (Texts)
> (15 to 30 students per class — Ijaza (certification) issued upon completion)

- Tuhfat Al-Atfal
- Al-Jazariyya
- Al-Shatibiyya

### 4. Arabic Language
- Al-Ajroomiyya
- Qatr Al-Nada

### 5. The Holy Quran
- **Memorization (Tahfeez)**
- **Revision (Muraja'a)**
  - Mateen track (revision of 10, 15, or 30 parts)
  - Thabbitaha track

### 6. Himam Al-Qurani Program
> (Held every Saturday)

- 3-Juz branch
- 5-Juz branch
- 10-Juz branch
- 15-Juz branch
- 30-Juz branch
